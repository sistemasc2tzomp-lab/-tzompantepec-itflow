# ITFlow - Mesa de Servicios Municipal

Sistema de gestión de tickets para el H. Ayuntamiento de Tzompantepec.

## 🚀 Despliegue Rápido

### Vercel (Recomendado)

1. **Haz fork de este repositorio**
2. **Ve a [Vercel](https://vercel.com)**
3. **Importa tu repositorio de GitHub**
4. **Configura las variables de entorno** (si es necesario)
5. **Haz clic en Deploy**

### Configuración Post-Despliegue

1. **Configurar Supabase**:
   - Ve a tu proyecto desplegado
   - Configura las credenciales de Supabase en la sección Configuración
   - Ejecuta el script de setup para crear usuarios iniciales

2. **Usuarios Iniciales**:
   ```
   Admin: admin@tzompantepec.gob.mx / admin123
   Usuario: carlos.hernandez@tzompantepec.gob.mx / user123
   ```

## 📋 Características

- ✅ Gestión completa de tickets
- ✅ Dashboard con métricas en tiempo real
- ✅ Sistema de roles (Admin/Usuario)
- ✅ Exportación a Excel y PDF
- ✅ 100% Responsive
- ✅ Kanban visual
- ✅ Reportes avanzados

## 🛠️ Stack Tecnológico

- **Frontend**: HTML5, CSS3, JavaScript Vanilla
- **Backend**: Supabase (PostgreSQL + Auth)
- **Charts**: Chart.js
- **Export**: jsPDF, SheetJS
- **Despliegue**: Vercel

## 📱 Acceso

Una vez desplegado, el sistema estará disponible en la URL que Vercel te asigne.
